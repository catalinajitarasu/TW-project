FoSO- Food Web Smart Organizer



FoSO (Food Web Smart Organizer) este o aplicatie Web care realizeaza managementul preferintelor unei persoane sau a unui grup de persoane in ceea ce priveste consumul alimentelor. Aceasta ofera suport si pentru gestiunea altor datelor aferente precum categorie, pret, ingrediente, imagini, restrictii (e.g., alergeni, regim special) si altele
